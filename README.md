## Installation ⚙️

1. Clone the repository:
   ```bash
   git clone https://github.com/airdropbomb/EnergyLabs.git && cd EnergyLabs
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Enter Your Account Detail
   ```bash
   nano accounts.json
   ```

4. Run Bot
  ```bash
  npm start
  ```
